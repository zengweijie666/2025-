# 图片目录

此目录用于存放网站所需的图片资源，包括：

1. logo.png - 网站徽标
2. graduation-hero-mobile.jpg - 首页移动端背景图
3. graduation-video.mp4 - 首页背景视频
4. featured-photo.jpg - 首页推荐照片
5. video-thumbnail-1.jpg, video-thumbnail-2.jpg - 视频缩略图
6. gallery-1.jpg 至 gallery-12.jpg - 照片墙照片
7. timeline-1.jpg 至 timeline-14.jpg - 时光轴照片
8. avatar-1.jpg 至 avatar-4.jpg - 用户头像

## 注意事项

- 请确保所有图片的版权合法
- 建议对大图进行压缩，以提高网站加载速度
- 照片分辨率建议不低于1200px宽度（照片墙展示用）
- 头像图片应为正方形，建议尺寸200x200像素

## 真实环境部署

在真实环境中部署时，请将本目录中的占位图片替换为班级的实际照片。
